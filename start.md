# Antigravity を始めよう

SuperAntigravity がインストールされました。準備は完了です！

## 次のステップ

1. チャットパネルで新しい会話を開きます。
2. **"hello"** と入力して、SuperAntigravity に挨拶してください。システムが利用可能であることを確認し、利用可能な機能の概要をお伝えします。

または、以下のコマンドを入力して機能をお試しください:

- `/brainstorm プロジェクトのアイデア` - 新しいものの構築を開始します
- `/research Antigravity の最新機能` - 調査を実行します
- `/recommend` - あなたの現在のコンテキストに基づいてコマンドを提案します

作業プロセスについては、[README](README.md) を参照してください。
